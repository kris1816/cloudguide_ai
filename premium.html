ℹ️ How It Works

Server-Side Processing: All API calls are made securely through your Vercel backend. Your API keys are stored as environment variables in Vercel, never exposed to the browser.

Setup Required:
1. Add CLAUDE_API_KEY to your Vercel environment variables
2. Add SPEECHIFY_API_KEY for premium voices (optional - falls back to OpenAI TTS)
3. Deploy the API endpoints (/api/claude.js and /api/speechify.js)
